C -  _printf

Project Description:

We made a clone of printf that use a function that prints a formated string.

How it works:

It produces output according to a format. 

Flags used:

It uses differents format specifers, each one does a different thing.

  <tr>

   <tr>

   <tr>
    <td>'%'</td>
